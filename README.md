# Monop'EFREI

## Projet:
Le projet consiste à recréer un Monopoly basé sur l'EFREI, dans lequel on peut jouer en multijoueur. Il sera fait à l'aide de HTML, CSS, Js, JSON et PHP. Vous travaillerez sur différentes branches et demanderez à merge avec la branche principale lorsque vous avez fini d'effectuer votre tâche.

## Règles:
- Utilisation de l'IA: Uniquement pour corriger ses erreurs, et le faire à modération, préférer utiliser des sites comme Reddit ou StackOverflow pour trouver des réponses. Le but de ce projet est de créer un jeu nous-même en y mettant du notre, et cela commence en écrivant le code sois-même. Il nous permettra de plus d'ajouter un projet à notre CV. C'est pourquoi nous avons décidé d'interdir l'utilisation de l'IA pour ce projet.
- Rester sérieux dans le travail: Ce projet reste sérieux, donc on évite de mettre n'importe quoi dedans, on reste dans le thème quoi qu'il en soit.
- On ne fait rien sans se concerter: Si vous décider de n'en faire qu'à votre tête et que ça ne plaît pas à tout le monde vous aurez perdu votre temps, donc proposez avant de faire. Il serait d'autant plus dommage que deux personnes travaillent en même temps sur la même chose.

## Contenu:
- Les lieux seront basés sur les batiments, personnes ou associations de l'EFREI.
- Les illustrations seront des photos prises sur les lieux.
- Les règles seront les règles d'un Monopoly classique, avec un plateau basé sur celui original, mais il pourra peut-être être créé aussi en version plus grande pour des parties à plus de joueurs.
